Hvað gera eftirfarandi Linux skipanir?
 cd - change directory
 ls - það þyðir list. Það skoðar hvað directory/folders sem þu ert að vinna með
 pwd - password
 mkdir - make directory
Hvað gera eftirfarandi Git skipanir?
 git clone - það downloadar git link sem copyaðir
 git status - sýnir working tree status
 git diff - sýnir munn á commits
Hvað gera eftirfarandi Git skipanir saman? Hvaða gagn er að þeirri aðgerð?
 git log - sýnir commit log
 git checkout - skiptir um greinar eða restorar tree files
 git branch -búa til eða eyða greinar
Hvað er útgáfustýring (Version Control)? - það breytir t.d Linux code yfir í windows code.
 Hverjir eru helstu kostir við að nota GIT? - Git gerir það auðveldara að vinna með oðrum
 Hversu oft telur þú að eigi að gera færslur (commit) í verkefni? - eins oft og maður vil 
 Hvað er átt við með “Working Directory”?” og “Staging Area” í GIT? - Working Directory er þaða directory sem þu er að vinna í en Staging Area er staður til að geyma data
